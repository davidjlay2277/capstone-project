Server

  • install nmp and axios
  • set up express and cors
  • set up server to listen on desired port
  • run with nodemon

///////////////////INITIALIZE SERVER ////////////////////
install nodemon (nmp install -g nodemon) run the following commands: npm init -y package.json installed npm i express package-lock.json installed npm i cors npm install dotenv Create a file in the root called.env Create a SERVER_PORT variable Create a CONNECTION_STRING variable (currently using supabase) ctrl.js, require in the dotenv package and call its config method (you can ignore the other code in there until later steps) npm install sequelize pg pg-hstore controller.js, require the sequelize package and save it to a variable called Sequelize Initialize a new Sequelize instance and pass in your connection string. nodemon run index js, spin up server then create git repository (see .gitignore for node_modules)
